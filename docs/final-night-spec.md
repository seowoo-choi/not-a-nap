# 백일째 밤 스펙 (원본)

이 문서는 백일째 밤(최종 보스전)의 규칙 원본이다. Reference/prototype.html에 없는 신규 규칙이며,
수치·공식을 임의로 바꾸지 말 것. 1~2일차 밸런스 원본은 prototype.html의 `<script id="game-logic">`.

## 인트로
"시간이 흘렀다. 오늘은 백일째 밤." + 지금까지 형성된 습관 요약 카드.

## 제약
- 아이템 슬롯 3개 → **2개**
- **할머니 찬스 사용 불가** (오늘 밤은 가족 없이)
- 기억(memory) 효과 계수 **1.5배** 적용 (습관의 부작용·이점 모두 증폭)

## 습관 표적 이벤트
형성된 습관에 따라 결정론적으로 선택, 최대 2개 발동:
- `memory.carrier ≥ 0.3` → 00시 "아기띠 버클 고장" (2턴간 아기띠 사용 불가)
- 백색소음 의존(`noiseHab ≥ 0.3` 또는 1~2일차 `noiseTurns` 합 ≥ 6) → 01시 "배터리 방전" (그 밤 백색소음기 꺼짐)
- `memory.heldDep ≥ 0.3` → 03시 "새벽 각성" (수면 -30, 눕히기 추가 페널티 +0.1)
- `memory.selfSoothe ≥ 0.3` → 깰 때마다 50% 확률로 스스로 재입면 (관찰·탈의존 플레이 보상)

## 승리 조건
아침 6시 기준 3개 중 **2개 이상**:
1. 아기가 깊은 잠(`sleep ≥ 85`) 상태로 아침을 맞음
2. 보호자 체력 ≥ 30
3. 그 밤에 "맨손 눕히기" 성공 1회 이상 (아기띠·바운서 미착용 상태에서 눕히기 성공)

## 클리어 연출
조건 충족 상태로 마지막 시간에 도달하면 화면에 처음으로 정확한 시각 "05:58" 표시
→ 아기가 뒤척이다 스스로 잠드는 대기 연출 → 아침 → "육아가 끝난 것은 아닙니다. 하지만 이 밤은 끝났습니다."

## 엔딩 6종 판정 (위에서부터 우선 적용)
1. **아침이 이겼다** (실패): 승리 조건 0~1개 — 아기가 말똥한 눈으로 일출 감상
2. **우리 집의 루틴**: 승리 + 모든 memory 의존치(carrier, heldDep, noiseHab) < 0.4 — 이상 엔딩
3. **품 안의 우주**: 승리 + carrier 또는 heldDep ≥ 0.5 — 품에서는 최강, 내려놓기는 숙제
4. **할머니가 최고야**: 승리 + 1~2일차에 할머니 찬스 사용 — 체력은 최고, 루틴은 가족마다 제각각
5. **장비의 지배자**: 승리 + 3일간 서로 다른 아이템 4종 이상 사용 — 자동화의 왕, 고장에 취약
6. **새벽의 생존자**: 그 외 승리 — 루틴은 없었지만 아침은 왔다
